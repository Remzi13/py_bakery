# Multi-Language Support

Система поддерживает английский (English) и русский (Русский) языки.

## Использование

### Переключение языков

1. **На странице**: В нижней части боковой панели есть два кнопки для переключения языков:
   - `English` - переключиться на английский
   - `Русский` - переключиться на русский

2. **Автоматическое восстановление**: Выбранный язык сохраняется в `localStorage` браузера и будет восстановлен при следующей загрузке страницы.

### Архитектура переводов

- **Файл переводов**: `static/js/translations.js`
- **Система**: Используется объект `translations` с ключами для каждого языка
- **Функция перевода**: `t(key)` - получить перевод по ключу
- **Переключение**: `switchLanguage(lang)` - переключить язык

### Добавление новых переводов

1. Откройте `static/js/translations.js`
2. Найдите нужный ключ в объектах `en` и `ru`
3. Добавьте или обновите перевод

Пример:
```javascript
en: {
    myNewKey: "English text",
    ...
},
ru: {
    myNewKey: "Русский текст",
    ...
}
```

### Использование переводов в HTML

Добавьте атрибут `data-i18n` к элементу:

```html
<button data-i18n="myKey">Default Text</button>
<input placeholder="text" data-i18n="myKey">
```

### Использование переводов в JavaScript

```javascript
const text = t('myKey');  // Получит перевод на текущий язык
```

## Поддерживаемые языки

- **Английский (en)** - язык по умолчанию
- **Русский (ru)** - полная локализация
